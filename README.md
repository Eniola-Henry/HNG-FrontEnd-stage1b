# HNG Frontend Stage 1B – Profile Card

A responsive, accessible Profile Card built using semantic HTML, modern CSS, and vanilla JavaScript.  
This project focuses on **structured UI design, accessibility compliance, and testability using data-testid attributes**.

---

## 🚀 Live Demo
https://eniola-henry.github.io/HNG-FrontEnd-stage1b/

---

## 📌 Project Overview

This Profile Card displays a user profile with:
- Avatar image
- Name
- Short biography
- Live epoch time (milliseconds)
- Social media links
- Hobbies
- Dislikes

The component is fully responsive and adapts across mobile, tablet, and desktop screens.

---

## 🧠 Features

- Semantic HTML structure (`article`, `nav`, `section`, `figure`)
- Live epoch time updated in real-time
- Accessible avatar with alt text
- Keyboard navigable social links
- Responsive layout (mobile → desktop)
- Clean visual hierarchy
- Testable UI using strict `data-testid` attributes

## 🛠️ Tech Stack

- HTML5 (Semantic Markup)
- CSS3 (Flexbox + Responsive Design)
- Vanilla JavaScript (DOM + Time updates)

---

## ♿ Accessibility

- Meaningful alt text for avatar
- Keyboard accessible links
- Proper semantic structure
- aria-live used for time updates
- High contrast UI for readability

---

## 📱 Responsiveness

- Mobile (320px+): stacked layout
- Tablet (768px+): balanced spacing
- Desktop (1024px+): side-by-side layout

No horizontal overflow at any screen size.
